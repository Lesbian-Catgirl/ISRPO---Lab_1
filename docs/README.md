# Math formulas
## Math formulas is a library for calculating areas and perimeters for diffenrent geometric figures: circle, rectangle, triangle, square

# Figures

## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
